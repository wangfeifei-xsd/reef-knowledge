# 海水鱼常见疾病大全（7-8）

本篇收录海水观赏鱼常见疾病，包括 HLLE、布鲁克原虫病、单侧突眼、双侧突眼及尾丝虫“渗血红斑”等典型病症。每种疾病均提供病原识别、症状诊断与治疗方案，供参考主对照鱼只实际情况进行判断与处理。

---

## HLLE（头部和侧线侵蚀）

HLLE（Head and Lateral Line Erosion）是海水观赏鱼中常见的一种慢性、退行性病变，主要表现为头部及侧线感官孔出现对称性凹陷、褪色、组织坏死，呈坑洞状。与大多数鱼病不同，HLLE 不是由细菌、病毒或寄生虫引起的传染病，而是一种生理性病变。

### 病原与诱因

HLLE 并非由于寄生虫或细菌引发，目前对其发病原因并未有明晰的结论，但根据养殖经验，其核心诱因可能包括：

| 诱因类型 | 具体说明 |
|---------|----------|
| 粉尘刺激 | 劣质活性炭释放的微粉末会刺激鱼类的侧线感官系统造成坏死 |
| 营养代谢障碍 | 缺乏关键微量元素和维生素，导致上皮细胞失去修复能力 |
| 漏电刺激 | 水体中的感应电压长期干扰侧线神经丘，引发组织凋亡 |
| 水质恶劣 | 水中较高的硝酸盐（NO₃⁻）浓度可能会对鱼产生压力，导致问题 |

### 易感鱼种

HLLE 常见于代谢活跃且侧线系统发达的鱼类：

- **刺尾鱼科**：典型代表为紫吊、蓝吊、黄金吊等
- **刺盖鱼科**：无论是大仙（刺盖鱼属）还是小仙（刺尻鱼属），也都易发现 HLLE

### 症状表现

HLLE 的演变通常是一个缓慢的过程，分为三个阶段：

**初期**：眼睛周围或额头出现细小的浅色斑点，类似针尖大小的凹陷，极易被误认为蹭伤。

**中期**：头部凹陷扩大，连接成不规则的坑洞。身体中部的侧线开始出现白色或浅色的长条状侵蚀带。

**后期**：组织深度坏死，甚至露出下方的骨骼，鱼只颜色黯淡，虽然仍能进食，但状态下降明显。

### 治疗方法与流程

由于 HLLE 不是感染性疾病，使用抗生素或驱虫药无效。建议做如下操作，排查诱因，促进鱼只恢复：

**1. 切断诱因**

- 立即停止使用任何易碎、有粉尘的劣质活性炭
- 使用万用表测量缸内电压（一端接地、一端伸入水中），如果有明显漏电，需排查具体漏电设备并进行更换
- 换水，勤洗滤棉，增强蛋分，维持水中硝酸盐浓度较低（NO₃⁻ 低于 20ppm）

**2. 增强营养**

- **优质饲料**：建议将饲料更换为营养成分全面，有充足维生素、微量元素添加的优质饲料，停用廉价三无饲料
- **增加新鲜藻类**：每日提供天然紫菜或海苔等，补充颗粒饲料中缺乏的矿物质和维生素

### 总结

通过早期的干预，大部分轻度 HLLE 是可以恢复的，但是如果侵蚀较深的晚期症状，可能无法完全恢复，导致鱼类“毁容”。HLLE 虽然不具传染性，却极度考验饲主的管理水平。治疗 HLLE 没有捷径，但只要干预及时，大多数鱼只的坑洞可以在优秀的水质中逐渐平复。

## 布鲁克原虫病（白膜病，白色粘膜）

布鲁克原虫（Brooklynella hostilis，俗称“小丑鱼白膜病”）是海水养殖中一种致病性强、病程迅猛的纤毛类寄生原虫。其繁殖周期短，病情发展快，若未及时干预，可在 24-48 小时内导致高死亡率，堪称鱼缸中的“无声火灾”。

### 病原体特征

**分类与形态**：属纤毛门，肾形或心形，体表覆盖纤毛，运动活跃，如同长着密集纤毛的微型“冲锋舟”。

**生活史**：以直接二分裂方式在宿主体表或鳃部快速繁殖，能在水体中自由游动并寻找新宿主，此阶段可延续数日至数周。这种生活史导致布鲁克原虫病程发展迅速、传染性高。

### 易感鱼种

虽可感染多种海水鱼类，但以下种类风险尤高：

- **小丑鱼**（雀鲷科双锯鱼属）：堪称该病的“头号目标”，感染率与死亡率最高
- 其他常见易感种类：海金鱼、刺尾鱼、蝴蝶鱼及神仙鱼等也属高发群体
- 特殊易感类群：海马体表防御较弱，感染后病情往往更为严峻

### 感染症状

**体表粘液增生（核心特征）**：体表（尤其是躯干、背鳍和尾柄部位）覆盖上一层浓厚、不透明、灰白至蓝白色的粘液层。这层粘液并非单纯水样分泌物，而是由脱落的上皮细胞、炎性细胞和原虫大量聚集构成。外观上，鱼仿佛披了一件粗糙的白膜，这也是“白膜病”俗称的由来。

**异常呼吸**：鳃部受损导致呼吸极度急促，鳃盖开合快速且幅度大（“喘气”状）。

**表皮损伤、衰竭**：严重时粘液脱落，伴随表皮侵蚀或溃疡，可能会伴随蒙眼情况；鱼体消瘦，脊柱可能显形。

**行为异常**：包括擦身、躲藏、食欲减退及反应迟钝。

### 治疗方法与流程

**注意**：常规铜制剂对该虫无效。

**1. 甲醛（福尔马林）药浴**

甲醛分子能迅速使蛋白质变性、凝固，破坏原生动物运动器官和体表结构，对于原生动物治疗效果非常突出。配合孔雀石绿，会增强其效果。

治疗布鲁克原虫感染，建议将鱼捞至 AB 桶，每 48 小时进行福尔马林 + 孔雀石绿药浴一次，至少三次。药浴同时消毒旧桶，药浴结束后将鱼放入新鲜海水的新桶中。观察直到白膜完全消失方可停止。

**2. 替代药物：磷酸氯喹**

磷酸氯喹也适用于对抗原生动物感染，对鱼体刺激相对较小，可以配合福尔马林药浴使用。可参考磷酸氯喹 HTTM 检疫法设置磷酸氯喹浓度。

### 原缸处理

由于布鲁克原虫的强传染性，想要根治，如下步骤不可或缺：

- **移除所有鱼类**：将原缸内所有鱼类转移至检疫缸治疗
- **维持空缸期**：原缸（含活石、底砂、设备）必须保持完全无鱼状态至少 4 周（建议 6-8 周），如同执行一场“隔离封锁”，使游离虫体因无宿主而自然消亡
- **勿依赖药物净缸**：不建议在原缸中直接使用甲醛等药物治疗，以免破坏系统平衡

### 检疫建议

所有新鱼入缸前应经过严格的检疫步骤，避免引入病原体，本程序介绍的磷酸氯喹 HTTM 检疫法，非常适合小丑鱼等易感鱼类。布鲁克原虫病的防控，需要针对病鱼的精准治疗、对原缸的彻底净化，以及对引入新鱼的严格检疫三者协同。任何环节的疏漏都可能导致疾病爆发。

## 单侧突眼

如果你的海水鱼只有一只眼睛凸出（单侧突眼），请不要慌张。这通常是物理损伤所致，如撞击缸壁、被装饰物刮伤或遭到其他鱼攻击。这种情况的核心处理原则是“少折腾，养水质”，让鱼儿依靠自身免疫力恢复。

### 原因与判断

**主要原因**：局部物理损伤导致眼部后方组织发炎、积液，将眼球向前推挤。

**关键特征**：仅一只眼睛明显凸出，鱼儿其他方面（游姿、食欲、体色）通常无明显异常。

**简单自检**：回忆一下鱼缸内是否有打架现象，或检查造景是否有尖锐处。

### 核心处理步骤（无需急于用药）

1. **保持优质水质**：这是恢复的基石。确保氨氮为 0，海缸水没有出现白色浑浊（生物性白浊），加强换水频率，使用蛋分、过滤系统维持水体清洁

2. **使用 UV 杀菌灯**：每天开启 12 小时，能有效降低水中游离的细菌数量，预防伤口发生继发感染

3. **考虑隔离静养**：如果鱼只因弱势常被攻击，将其放入缸内大隔离盒，可以减少压力、避免二次伤害，并提供一个更安静的环境恢复

4. **耐心观察**：避免频繁捞鱼、下药等过度干预行为，这些会增加鱼的压力，反而不利于恢复

### 恢复预期与注意事项

- **恢复时间**：轻微的突眼可能在 1 周到数周内自行消退。如果凸出严重，眼球可能无法完全复位到原始状态

- **重点观察**：只要鱼儿精神状态良好、主动进食，就应保持耐心。持续观察眼部是否出现恶化（如浑浊、溃烂）或出现双侧凸出等新症状。若出现恶化，则要考虑细菌感染，进行相应的抗生素干预

- **关于治疗**：不推荐对此情况使用抗生素（如黄粉、各种“沙星”），因为无效且可能破坏水质菌群。对于极高价值的鱼只，眼球严重凸出时，专业水族机构可能会采用“加压舱”处理，但这不适用于家庭新手

### 总结清单（单侧突眼）

| 项目 | 内容 |
|------|------|
| 判断 | 单眼凸出，大概率是撞伤 |
| 行动 | 优化水质（测氨氮、开 UV 灯）→ 隔离静养 → 耐心观察 |
| 避免 | 乱下药、频繁惊扰鱼儿 |
| 关键 | 时间是良药，鱼的自我修复能力是关键 |

## 双侧突眼

当你的海水鱼两只眼睛同时凸出（双侧突眼），这是一个**需要立即警惕的信号**！这通常不是单纯的眼部问题，而是全身性细菌感染的严重表现之一。必须采取果断、系统的医疗措施。

### 原因与严重性

**核心原因**：细菌通过鳃、体表伤口等侵入鱼体内部，引发全身性炎症和液体积聚，眼部症状只是“冰山一角”。

**伴随症状**：常同时出现蒙眼、呼吸急促、鳍条溃烂、体色异常、食欲减退等。

**紧急程度**：高。需要立即干预，否则可能危及生命。

### 核心处理步骤（必须系统治疗）

**第一步：立即隔离**

将病鱼转移至准备好的检疫缸（“治疗缸”）。该缸应水质洁净、有独立过滤和增氧设备。这是防止感染扩散和进行精准治疗的关键。

**第二步：强化水质管理**

每日大换水：在检疫缸中，每天换水 80% 以上，最大限度降低水中的细菌负载和氨氮毒素，为鱼体恢复创造最佳水体环境。

**注意**：如果使用抗生素治疗期间，请关闭 UV 杀菌灯，因为紫外线会分解多数抗生素，导致失效。

**第三步：正确进行抗生素治疗（核心环节）**

全身感染必须使用抗生素，并确保药物能进入鱼体内。

**给药方式**：

| 给药方式 | 推荐药物 | 说明 |
|----------|----------|------|
| 水中投药 | 紧急 B 溶液 | 按说明剂量及使用方法，疗程 5-7 天。此操作最简单，适合大多数新手。药物在海水中稳定，能被鱼体吸收 |
| 食物投药 | Enro 颗粒 | 或使用专业凝胶粉制作药饵。喹诺酮类药物口服效果更好，前提是鱼还能主动进食，确保药物被吃进去 |
| 注射给药 | 观赏鱼专用阿米卡星或恩诺沙星注射液 | 需在麻醉下按体重精准注射。效果最快最强，但仅限有丰富经验的鱼友操作，风险极高 |

### 治疗误区警示

- **追求“神速”**：抗生素不是消毒剂，需要作用时间，请保持耐心
- **“自治”偏方**：不要用人类或其他宠物药物，也不轻信非专业建议
- **忽视根源**：检查并处理可能引发细菌感染的原发病因（如寄生虫、外伤）

### 总结清单（双侧突眼）

| 项目 | 内容 |
|------|------|
| 判断 | 双眼凸出，是严重全身感染警报 |
| 行动 | 立即隔离 → 检疫缸每日大换水 → 选择正确抗生素并满足全疗程治疗 |
| 避免 | 使用无效药物、提前停药、治疗期间开 UV 灯 |
| 关键 | 快速响应、系统治疗、坚持到底 |

## 尾丝虫“渗血红斑”（爆身）

养海水观赏鱼的朋友，大概率都听过或见过海金鱼、蝶鱼或是青魔身上突然出现渗血般的红斑，若鱼没能挺过这一关，渗血区域会在 1-2 天内迅速扩大，进而溃烂成深洞，甚至穿透鱼身。因其发展速度快、最终惨状像“肉被硬生生啃掉一块”，便有了“爆身”这个听起来颇为惊悚的俗称。

业余博主最爱的说法是“这是细菌感染”，但事实并非如此，造成此症状的病原体是一大类原生动物——**盾纤毛虫 / 尾丝虫**。

### 核心信息

| 项目 | 内容 |
|------|------|
| 症状 | 身体快速出现渗血红斑并溃烂，精神萎靡，离群 |
| 病原 | 盾纤毛虫 / 尾丝虫（原生动物，非细菌） |
| 治疗 | 福尔马林 + 孔雀石绿药浴（每 48 小时一次，至少 5 次） |
| 辅助治疗 | 药浴间隔期，在治疗缸使用磷酸氯喹或 Rally Pro |

### 认识疾病：正名与元凶

“爆身”这一名称在交流中极易造成混淆。从病原学角度，造成渗血红斑症状的元凶是盾纤毛虫（又称尾丝虫），属于纤毛类原生动物，而非细菌感染。这一要点对于选择正确的治疗方案至关重要——普通的抗生素类药物对原生动物无效，必须使用针对原虫的特异药物。
