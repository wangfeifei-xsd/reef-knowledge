## 一、新发现的珊瑚吞食性扁虫

在珊瑚水族贸易中，存在一类专以珊瑚为食的扁虫，对珊瑚养殖构成威胁。继AEFW（Acropora-eating flatworm，鹿角珊瑚吞食扁虫）之后，国内外又发现了一种新的珊瑚吞食性扁虫。

### 1.1 形态特征

- **寄主范围**：嗜石珊瑚属（如 *Favia*、*Favites*、*Goniastrea*）等
- **取食方式**：与AEFW一样从珊瑚基部开始啃食，留下白点状痕迹
- **卵块特征**：黄褐色、直径不到1mm的扁圆形，比AEFW的卵块更为分散

### 1.2 来源与传播

- 最早在印度尼西亚进口的珊瑚上发现
- 2018年印尼禁止石珊瑚出口后，中国国内几乎再无新感染案例，推测原产地为印尼
- 2019年美国鱼友也发现了该虫，传播速度较慢

### 1.3 杀灭方法

常规检疫手段（氯化钾、Coral RX）对该虫均无效。作者通过实验验证了以下方法的可行性：

| 步骤 | 操作方法 |
|------|----------|
| 药物配制 | 使用4.5%高效氟氯氰菊酯乳油，按“1滴/1升海水”的比例配制 |
| 浸泡处理 | 将感染珊瑚浸泡10分钟，期间摇动使虫体脱落 |
| 清洗 | 用干净海水涮洗数次 |
| 卵块处理 | 用阿隆发胶将底部的卵块封住 |
| 重复 | 每7天重复一次，连续数次以确保杀灭干净 |

> **注意**：高效氟氯氰菊酯为合成拟除虫菊酯，使用时需佩戴手套。

### 1.4 命名与后续

原本准备以女朋友名字或“蜡笔小新”命名该虫，但因师弟将活体样品养死而未能完成定种。印尼珊瑚出口禁令间接导致科研工作受阻，作者感叹生物学研究中的“蝴蝶效应”。

## 二、珊瑚疾病研究进展

### 2.1 研究背景

目前人类对珊瑚疾病的了解仍然非常有限。大多数情况下，我们只知道某些症状与某些病原可能存在关联，但具体致病机理尚未明确。相对而言，弧菌是少数有较深入研究的病原之一。

### 2.2 新发现的立克次氏体寄生虫

2019年发表于顶级期刊《The ISME Journal》的一项研究揭示了一种新的海水无脊椎动物相关立克次氏体寄生虫。

**研究发现**：

- 研究团队在鹿角珊瑚研究中发现，营养盐上升会导致珊瑚死亡率升高
- 测序分析显示，一种立克次氏体的丰度在高营养盐环境下显著升高
- 立克次氏体是一类细胞内专性寄生性原核微生物

### 2.3 基因组分析

该立克次氏体（*Candidatus Aquarickettsia rohweri*）的基因组特征：

- 基因组仅1.28Mb，大量基因功能缺失
- 碳、氮相关基因几乎全部不完整
- 氨基酸合成通路缺了一大半，只能合成丝氨酸、甘氨酸、谷氨酸、天冬氨酸、赖氨酸和苏氨酸
- 缺失的氨基酸依赖宿主提供，因此必须寄生
- Type IV Secretion System（T4SS）基因完整，是寄生的关键证据

### 2.4 生态分布与致病机制

- 显微镜染色和荧光原位杂交显示，该立克次氏体特别喜欢聚集在虫黄藻附近的珊瑚细胞中，推测是为了吸收虫黄藻释放的氨基酸
- 与公共数据库比对发现，该立克次氏体在全世界的珊瑚中广泛存在，不仅限于鹿角珊瑚，甚至在海绵中也能找到
- **致病机制**：平时在珊瑚细胞中寄生，条件好时珊瑚能控制其数量；当环境恶化、珊瑚免疫力下降时，它会大量增殖，耗尽宿主营养，导致珊瑚死亡

> 作者感慨：“且行且珍惜，你真的不知道你的珊瑚是什么原因死的。”

## 三、珊瑚共生功能体概述

珊瑚并非单一的生物体，而是与多种微生物共同构成“珊瑚共生功能体”（Coral Holobiont），其整体基因组被称为“共生功能基因组”（Hologenome）。

### 3.1 组成成员

| 成员类型 | 说明 |
|----------|------|
| 虫黄藻 | 主要能量来源，通过光合作用提供有机碳 |
| 细菌 | 参与营养循环、防御病原 |
| 古菌 | 可能参与氮循环 |
| 真菌 | 部分可在骨骼上打孔栖息 |
| 顶复门原虫 | 新发现的共生原生动物，约70%珊瑚携带，功能尚不明确（具有不完整的叶绿素基因，不能光合作用） |
| 纤毛虫 | 以细菌为食，与骨骼侵蚀带病、褐色带病有关 |
| 病毒/类病毒颗粒 | 广泛存在，可能与白化和疾病有关 |

### 3.2 空间结构分布

| 部位 | 微生物特征 |
|------|-------------|
| 表面黏液层 | 每立方厘米含100万–1亿微生物，具保护作用 |
| 上皮与内胚组织 | 含有高度组织特异性的微生物聚集体，如河氏菌（*Endozoicomonas*）的囊状聚集体 |
| 肠腔 | 极低氧、高营养盐环境，维持肠道微生物稳定 |
| 骨骼 | 为丝状藻、细菌、真菌提供栖息地 |

### 3.3 物质循环

**碳循环**：虫黄藻光合作用提供主要能量；细菌和古菌也参与固碳和碳降解。

**氮循环**：微生物通过固氮、硝化、反硝化调节氮平衡。固氮需无氧环境。

**硫循环**：珊瑚是DMSP（二甲基硫丙酸）的主要生产者之一，DMSP吸引共生微生物；硫酸盐还原菌与黑带病有关。

**磷与其他元素**：微生物可降解有机磷提供磷源；提供B族维生素等必需物质。

### 3.4 环境压力影响

- **温度**：升温导致虫黄藻排出（白化），同时机会型病原菌增多
- **酸化**：可能促进固氮和初级生产，但不同珊瑚反应不同
- **底泥与营养盐**：有机物增加导致异养菌繁殖、缺氧、硫化物积累
- **累积效应**：过度捕捞+富营养化+升温，协同削弱珊瑚恢复力

> 作者特别提醒：“看看就行了，别想着去指导怎么养珊瑚，共生微生物群落的复杂性是远远超乎你们想象的。”

## 四、珊瑚荧光蛋白与“灯塔假说”

### 4.1 研究背景

珊瑚的荧光蛋白功能一直是学界关注的问题。在博士入学前，作者曾想研究珊瑚荧光蛋白功能，但导师和师兄都建议不要做这个方向，称之为“death trap”。最终作者放弃该方向，转向气候变化研究。

### 4.2 主流假说

- **光保护假说**：荧光将短波光转化为长波光，部分能量转为热能，避免光损伤
- **生产力假说**：深水珊瑚将蓝光转化为其他波长的光，提高光合效率
- **抗氧化剂假说**等

### 4.3 “灯塔假说”的提出与发展

| 年份 | 里程碑事件 |
|------|-----------|
| 1999年 | Horiguchi等发现自由生活的虫黄藻对特定波长有趋光性 |
| 2004年 | Hollingsworth等发现石芝珊瑚幼体在口部周围高表达绿色荧光蛋白，提出“灯塔假说”——绿色荧光像灯塔一样引导虫黄藻前来共生 |
| 2005年 | Hollingsworth等将白光分散成不同颜色照射虫黄藻培养容器，15分钟后虫黄藻聚集在绿光区域，初步验证假说 |
| 2019年 | Yusuke等在PNAS发表研究，进一步验证该假说 |

### 4.4 2019年PNAS研究验证

**实验设计**：

- 将纯培养虫黄藻放入一端有光源、一端黑暗的容器，计算趋光性系数
- 结果显示：虫黄藻最喜欢中等光强下的绿光
- 虫黄藻在光照12小时后（近天黑）趋光性消失，与夜间不运动习性一致

**关键实验**：

- 使用有绿色荧光的活珊瑚（对照为死亡珊瑚骨）和涂有绿色荧光蛋白的塑料块（对照为未涂的塑料块），放入含虫黄藻的水中
- 结果：有绿色荧光的活珊瑚和涂有荧光蛋白的塑料块均吸引更多虫黄藻，且只有在蓝光下这种差异才明显
- 将涂有荧光蛋白的塑料做成“虫黄藻陷阱”投入海中，同样捕获更多虫黄藻

### 4.5 未解问题

- 成体珊瑚不需要从环境获取虫黄藻，为何还大量表达荧光蛋白？
- 很多荧光蛋白并非表达在口部周围
- 很多珊瑚的荧光不是绿色，而是红、黄等色
- 非共生的虫黄藻为何也对特定波长有趋光性？
- 很多珊瑚根本没有虫黄藻

> 总结：荧光蛋白的作用可能是多种多样的。

## 五、虫黄藻的多样性与分类学修订

### 5.1 虫黄藻的多样性

珊瑚是一个庞大的家族，而虫黄藻同样是一个庞大的家族。共生虫黄藻传统上被归入 *Symbiodinium* 属，分成A–I九个系群（clade），之下还有亚系群、株系。真正被定种的只有22种，潜在物种却有几百种之多。

每颗珊瑚都有自己独特的虫黄藻组成，甚至同一珊瑚不同部位也有差异。虫黄藻组成不同，对珊瑚生理功能和环境响应有很大影响。

### 5.2 区分方法

- **大小**：不同系群大小可差几倍甚至几十倍
- **形态**：在行家眼中，不同虫黄藻形态各异，尤其clade C较易区分
- **分子手段**：最主要、最可靠的分类方法

### 5.3 分类学大修订（2018年）

Todd LaJeunesse团队在2018年发表于 *Current Biology* 的研究中，对虫黄藻分类学进行了重大修订：

**起源时间**：分子钟测定显示，虫黄藻最早分化始于约1.6亿年前的中生代，远早于先前估计。早期的共生珊瑚在侏罗纪末期经历过一次灭绝。

**分类层级提升**：原 *Symbiodinium* 属被提升为 *Symbiodiniaceae* 科，原有的九个系群（clade）被提升为属。

**新命名的七个属**：

| 原 clade 名称 | 新属名 |
|--------------|--------|
| clade A | *Symbiodinium*（保留原名） |
| clade B | *Breviolum* |
| clade C | *Cladocopium* |
| clade D | *Durusdinium* |
| clade E | *Effrenium* |
| clade F | *Fugacium* |
| clade G | *Gaduspium* |
| clade H | *Herreosulcus* |
| clade I | *Insolatas* |
